# Инварианты и критерии отбора — проект для утверждения

## Зачем этот документ

Формализация правил, по которым Plot Bot автономно фильтрует и оценивает участки. Без утверждённых инвариантов агент не может принимать решения о приоритизации. Каждый пункт ниже требует подтверждения или корректировки от Вадима.

---

## Часть 1: Инварианты бизнеса

Нарушение любого инварианта = полная остановка + эскалация.

### И1. Визуальный контроль окружения
> «В зоне видимости ничего некрасивого»

**Текущая интерпретация:**
- Радиус 500м от границ участка — нет промышленных объектов, свалок, руин, заброшенных строений
- Все строения в кластере — только через Архнадзор (пул утверждённых архитекторов)
- Визуальная оценка: Pristine (1.0), Mixed (0.5), Degraded (0.1 — reject)

**Вопросы Вадиму:**
- [ ] 500м — правильный радиус? Или шире / уже?
- [ ] Что считается «некрасивым»? Конкретный список (заброшенные стройки, промзоны, мусор, ветхие дома, ЛЭП, вышки)?
- [ ] Допускаются ли участки с «некрасивым» окружением, если окружение можно контролировать (скупить/снести)?

---

### И2. Архнадзор
> «Все стройки только через Архнадзор»

**Текущая интерпретация:**
- При продаже участка — обременение: строительство только по проекту из пула Архнадзора
- Стоимость архпроекта: ~$3,000 за участок (заложено в unit economics)
- Юридическая форма обременения — исследуется (Phase 3)

**Вопросы Вадиму:**
- [ ] Архнадзор = конкретные архитекторы или стилевой гайдлайн?
- [ ] Обременение при продаже — нотариально? Через договор?
- [ ] Уже есть архитекторы в пуле?

---

### И3. Положительный кэш-баланс
> «Никогда не уходить в минус»

**Текущая проблема:** Unit economics показывает **отрицательный баланс в месяцы 1-6** при схеме 30/70 (equity/debt). Это нарушает инвариант.

**Варианты решения:**

| Вариант | Equity | Debt | Cash gap | Плюсы | Минусы |
|---------|--------|------|----------|-------|--------|
| A. Pre-sales (2-3 участка по $150/м²) | 30% | 70% | Закрыт pre-sales | Сохраняет leverage | Скидка ~25% на pre-sale |
| B. Больше equity (40/60) | 40% | 60% | Сужен до ~$40K | Менее рисковый | Нужно больше своих |
| C. Транши кредита | 30% | 70% (в траншах) | Закрыт синхронизацией | Оптимальный leverage | Банк может не согласиться |
| D. Меньший участок (10,000 м²) | 30% | 70% | Закрыт масштабом | Меньше капитала | Меньше ROI |

**Вопросы Вадиму:**
- [ ] Какой вариант предпочтительнее? (рекомендация: C → A → B)
- [ ] Допустимо ли краткосрочное (1-2 месяца) отрицательное сальдо при наличии кредитной линии?

---

### И4. Покупка сильно ниже рынка
> «Минимум 5x потенциал роста»

**Текущие пороги (из scoring model):**
- Максимальная цена входа: **≤$40/м²** (фильтр F1)
- Целевая цена продажи: **$200/м²** (conservative) — **$350/м²** (optimistic)
- При $25/м² → потенциал 8-14x ✅
- При $40/м² → потенциал 5-8.75x ✅
- При $50/м² → потенциал 4-7x ⚠️ (пограничный)

**Вопросы Вадиму:**
- [ ] $40/м² — верный потолок для фильтра?
- [ ] 5x — это на equity или на total capital? (equity ROI vs total ROI)
- [ ] Целевая цена продажи $200/м² — откуда? Есть ли comparable sales?
- [ ] Каприви — $350/м²? Это за участок с домом или без?

---

### И5. Кластерный подход
> «Скупаем максимум в одной зоне, не распыляемся»

**Текущая интерпретация:**
- Минимальная площадь кластера: **нужно определить** (модель: 20,000 м² → 25 лотов по 800 м²)
- Один кластер за раз — не начинать второй пока первый не завершён
- Приоритет: смежные участки у малого числа владельцев

**Вопросы Вадиму:**
- [ ] Минимальный кластер: 10,000 м²? 20,000 м²? 5,000 м² для MVP?
- [ ] Размер лота: 800 м² — правильно? Или 500-1000 м² диапазон?
- [ ] Допускается ли «мини-кластер» (3-5 участков) для первого пилота?
- [ ] Целевой район для MVP: Григолети? Уреки? Другое?

## Часть 2: Критерии отбора участков (scoring model summary)

Полная модель: `work/scoring_model.md`. Здесь — ключевые параметры для утверждения.

### Фильтры (Pass/Fail)

| # | Критерий | Порог | Статус |
|---|----------|-------|--------|
| F1 | Цена входа | ≤$40/м² | ⚠️ Нужно подтверждение |
| F2 | Минимальная площадь | ≥3,000 м² | ⚠️ Нужно подтверждение |
| F3 | Расстояние до моря | ≤3 км | ⚠️ Нужно подтверждение |
| F4 | Юридическая чистота | Нет обременений | ✅ Логично |
| F5 | Назначение земли | Не с/х или перевод возможен | ✅ Логично |
| F6 | Визуальное окружение | Нет промышленных объектов в 500м | ⚠️ Радиус? |
| F7 | Подъездная дорога | Есть хотя бы грунтовая | ✅ Логично |

### Веса скоринга

| Категория | Вес | Что оценивает |
|-----------|-----|---------------|
| Ценовой потенциал | **30%** | Дисконт к медиане + потенциал роста |
| Локация | **25%** | Расстояние до моря, вид, окружение |
| Участок | **20%** | Площадь, форма, рельеф, грунт |
| Инфраструктура | **15%** | Дорога, электричество, вода |
| Кластерный потенциал | **10%** | Смежные участки, однородность владения |

**Вопрос Вадиму:**
- [ ] Веса адекватны? Или что-то пере/недооценено?
- [ ] Есть ли факторы, которые не учтены? (политические риски, соседи-конфликтники, сейсмика?)

## Часть 3: Пороги действий

| Score | Класс | Автоматическое действие |
|-------|-------|------------------------|
| 80-100 | A — Prime | Deep due diligence + **эскалация Вадиму** |
| 60-79 | B — Strong | Детальный анализ, pipeline |
| 40-59 | C — Moderate | Мониторинг |
| 20-39 | D — Weak | Только как часть кластера с A/B |
| 0-19 | F — Reject | Не рассматривать |

**Вопрос Вадиму:**
- [ ] При каком score эскалировать? (сейчас: ≥80, рекомендуем ≥60 для первых 50 участков — калибровка)
- [ ] Автономная покупка никогда — всегда через Вадима. Верно?
