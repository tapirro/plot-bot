# SS.ge API Access — Escalation

## Суть

SS.ge — крупнейший маркетплейс недвижимости Грузии (~40K DAU, ~550K MAU). У них есть **официальный API** (анонсирован на ss.ge/ka/blog/ss-ge-api--186). Для получения доступа нужно связаться с SS.ge напрямую — это **внешний контакт**, поэтому решение за Вадимом.

## Зачем нам API

1. **Данные о ценах.** У нас 28 прайсированных листингов с Place.ge для побережья. SS.ge даст на порядок больше — это #1 площадка страны. Без SS.ge наш scoring model работает на выборке, а не на генеральной совокупности.
2. **Мониторинг в реальном времени.** API позволит отслеживать новые листинги земли в целевых зонах — мы увидим underpriced участки раньше конкурентов.
3. **Калибровка scoring model.** Для калибровки весов нужно ≥50 оценённых участков. Place.ge дал 28. SS.ge закроет gap.
4. **Информационное преимущество** — инвариант миссии.

## Что запросить

| Пункт | Детали |
|-------|--------|
| **Endpoint** | Read access: поиск/фильтрация листингов земли по региону, цене, площади |
| **Данные** | Цена (GEL/USD), площадь (м²), координаты/адрес, тип объекта, дата публикации, фото |
| **Регион** | Аджара + Гурия (Батуми, Кобулети, Уреки, Григолети, Натанеби) |
| **Формат** | JSON API (REST) |
| **Rate limits** | Достаточно 100 запросов/день для мониторинга |
| **Стоимость** | Уточнить — может быть бесплатно для партнёров или платно |

## Варианты действий

### Вариант A: Официальный запрос API (рекомендуемый)
- Написать на info@ss.ge от лица Mantissa Lab
- Запросить документацию и условия доступа к API
- **Плюсы:** легальный, стабильный, полный доступ к данным
- **Минусы:** может быть платно, может быть долго (1-4 недели)

### Вариант B: Headless browser scraping
- Написать скрапер на Playwright (обходя SPA)
- **Плюсы:** не нужно ждать, сразу работает
- **Минусы:** хрупкий (ломается при обновлениях), может нарушать ToS, нагрузка на SS.ge

### Вариант C: Гибридный
- Запросить API (Вариант A) + временный скрапер (Вариант B) пока ждём доступ
- **Плюсы:** не теряем время
- **Минусы:** двойная работа

## Рекомендация

**Вариант A.** Официальный API — единственный надёжный путь для production-grade мониторинга. Скрапер — запасной, но лучше не начинать с нарушения ToS площадки, на которой мы потом можем размещать свои листинги.
