# Plot Bot — AI Real Estate Developer (Georgian Coastline)

## Концепт

AI-агент «девелопер» ($200/мес), который автономно строит полный real estate development бизнес на грузинском побережье. Работает 24/7, делает юридическую часть, инвестпроект, анализ земли, маркетинг, продажи.

## Дальний горизонт

1. **Земельный анализ всей Грузии** — каждый кадастровый участок с гипотезами стоимости. Факторы: река, соседние постройки, этажность, госземля, пляж, охранная территория, камни/песок, подъезд
2. **Инвестиционный пул** — постоянный приток новых инвесторов. На старте: основатели
3. **Юридическая подсистема** — отдельный суб-агент, контракты, обоснования
4. **Финансовая машинка** — банковские кредитные линии (TBC Concept и др.), арифметика окупаемости, положительный кэш-баланс
5. **Маркетинг** — сайты, лендинги, рассылки через банки-партнёры
6. **Архнадзор** — пул доверенных архитекторов, все стройки только через них. Часть денег от продажи земли сразу откладывается на оплату Архнадзора

## Стратегия закупки

- Покупать **сильно ниже рынка**: $20-30/м² (аналог качества Каправани по $350/м²)
- Целевой ROI: **10x**
- Скупать **кластерами**: найти недооценённый участок → скупить максимум вокруг → контроль вида
- Нарезка: купить 20,000 м² → нарезать 800 м² участками → продать часть → закрыть кредит

## Инварианты (принципы)

1. В зоне видимости **ничего некрасивого** — лес по периметру, парки, контролируемая застройка
2. Подъезд тоже красивый (по возможности скупить землю вдоль дороги)
3. Всё строится только через Архнадзор
4. Положительный кэш-баланс всегда
