# Plot Bot — Strategic Roadmap

## Гипотеза

Полноценный AI-driven девелоперский бизнес требует 6 фундаментальных систем, развёрнутых последовательно. Каждая фаза создаёт предпосылки для следующей. MVP — фаза 1 (Intelligence) на одном кластере.

---

## Phase 0: Стратегический фундамент

Без этого нельзя начинать ни одну техническую работу.

### 0.1 Бизнес-модель и Unit Economics
- [ ] Описать полный цикл: покупка → оформление → нарезка → облагораживание → продажа
- [ ] Рассчитать unit economics на модельном участке (20,000 м², $25/м²)
- [ ] Определить минимальный капитал для старта (equity + debt mix)
- [ ] Смоделировать cash flow для первых 12 месяцев
- [ ] Определить break-even (сколько м² нужно продать чтобы закрыть кредит)

### 0.2 Инварианты и принципы
- [ ] Формализовать инварианты (визуальные, архитектурные, финансовые)
- [ ] Определить критерии отбора участков (scoring model)
- [ ] Сформулировать критерии «красоты» — что именно допускается/не допускается
- [ ] Определить минимальный размер кластера

### 0.3 Юридическая структура
- [ ] Исследовать форму организации (ООО, SP, фонд?) по грузинскому праву
- [ ] Определить структуру инвестиционного пула (как входят/выходят инвесторы)
- [ ] Исследовать ограничения для иностранцев на покупку с/х земли
- [ ] Определить налоговые последствия (покупка, владение, продажа)
- [ ] Исследовать формат Архнадзора юридически (как обязать строить через пул архитекторов)

## Phase 1: Intelligence — Земельный анализ

Ядро системы. Создаёт информационное преимущество.

### 1.1 Источники данных
- [ ] Инвентаризация доступных источников (NAPR, ArcGIS, Mymarket, SS.ge, Place.ge, публичный реестр)
- [ ] Оценить покрытие каждого источника (какие данные, какие регионы, качество)
- [ ] Найти источники по ценам сделок (есть ли публичный реестр продаж?)
- [ ] Найти источники по зонированию и разрешённому использованию
- [ ] Исследовать данные по геологии/геодезии (уклоны, грунт, камни vs песок)
- [ ] Найти данные по инфраструктуре (дороги, электричество, вода, канализация)
- [ ] Найти спутниковые/аэро снимки (Google Earth, Sentinel, коммерческие)

### 1.2 Инструменты сбора данных
- [ ] Расширить napr_lookup.py — batch-запросы по районам
- [ ] Написать скрапер Mymarket.ge (земельные объявления)
- [ ] Написать скрапер SS.ge (земельные объявления)
- [ ] Написать скрапер Place.ge (если есть земля)
- [ ] Интеграция с ArcGIS для массового получения кадастровых данных
- [ ] Инструмент загрузки спутниковых снимков по координатам

### 1.3 Хранение данных
- [ ] Спроектировать схему данных (участки, факторы, оценки, история цен)
- [ ] Выбрать storage (SQLite → PostgreSQL? + PostGIS для геоданных?)
- [ ] Реализовать ETL: источники → единая база
- [ ] Версионирование оценок (история изменений цен)

### 1.4 Методология анализа
- [ ] Определить полный набор ценообразующих факторов
- [ ] Разработать scoring model для участков (взвешенная оценка)
- [ ] Определить веса факторов (расстояние до моря, вид, уклон, доступ, инфраструктура...)
- [ ] Методология кластерного анализа (как находить перспективные зоны)
- [ ] Методология оценки потенциала (текущая цена vs потенциал после развития)

### 1.5 Калькулятор ценообразования
- [ ] MVP: калькулятор стоимости участка по факторам
- [ ] Сравнение с аналогами (comparable sales)
- [ ] Оценка стоимости после облагораживания
- [ ] Оценка стоимости с домом (по типам домов)

### 1.6 Визуализация
- [ ] Карта с участками и оценками (Leaflet/Mapbox)
- [ ] Дашборд: портфель участков, P&L, pipeline
- [ ] Heatmap потенциала по регионам

## Phase 2: Финансовый движок

### 2.1 Финансовое моделирование
- [ ] Модель: покупка крупного участка → нарезка → продажа по частям
- [ ] Калькулятор кредитной нагрузки (ставка, срок, график платежей)
- [ ] Симулятор cash flow с несколькими кластерами параллельно
- [ ] Сценарный анализ (оптимистичный / базовый / пессимистичный)

### 2.2 Банковские партнёрства
- [ ] Исследовать кредитные продукты TBC, BoG, Liberty для земельных сделок
- [ ] Определить требования банков (залог, первый взнос, ставка)
- [ ] Подготовить pitch deck для банков
- [ ] Модель: несколько кредитных линий, ротация

### 2.3 Инвестиционный пул
- [ ] Механика входа/выхода инвесторов
- [ ] Расчёт долей и распределения прибыли
- [ ] Отчётность для инвесторов (автоматическая)
- [ ] Dashboard для инвесторов
